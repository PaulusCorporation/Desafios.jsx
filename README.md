# ⚛️ Desafios React — UNIESP

Projeto com os 5 desafios + bônus da disciplina de Front-end (Parte 3 do material).

## 🚀 Como rodar

1. Instale as dependências:
```bash
npm install
```

2. Em um terminal, suba o React:
```bash
npm run dev
```

3. Em outro terminal, suba o back-end fake:
```bash
npm run server
```

4. Acesse [http://localhost:5173](http://localhost:5173)

## 📋 Desafios

| # | Componente | Conceito |
|---|---|---|
| 1 | `MensagemAoAbrir` | Primeiro `useEffect` |
| 2 | `data/db.json` | json-server |
| 3 | `ListaProdutos` | `axios.get` |
| 4 | `CadastrarTarefa` | `axios.post` |
| 5 | `RemoverProduto` | `axios.delete` |
| 🎁 | `ListaRecados` | CRUD completo |

## 🛠️ Tecnologias

- React + Vite
- Axios
- JSON Server
